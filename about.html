About Me

My name is Lyra, and this project also serves as the foundation for my personal web presence at https://lyra-on.top. Beyond fulfilling the laboratory requirements, I aimed to produce a polished, practical, and visually appealing website. Careful attention was given to layout, readability, and consistent styling across all pages.

ITEC50 Lab #2: HTML Basic Tags with Forms

This exercise fulfills the requirements for ITEC50 Laboratory #2. The objective was to create a multi-page website using semantic HTML elements, navigation links, and interactive forms.

The project demonstrates the following skills and accomplishments:

  1. Structuring pages with semantic HTML5 elements (<header>, <nav>, <section>, <article>, <footer>)
  2. Creating functional contact and feedback forms
  3. Implementing headings, paragraphs, lists, and text formatting for clear readability
  4. Ensuring logical navigation across multiple pages
  5. Designing a visually consistent and responsive layout
