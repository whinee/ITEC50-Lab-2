About Me

My name is Lyra, and this project also serves as the foundation for my personal web presence at https://lyra-on.top. Beyond fulfilling the laboratory requirements, I aimed to produce a polished, practical, and visually appealing website. Careful attention was given to layout, readability, and consistent styling across all pages.

This exercise helped me strengthen my skills in semantic HTML, structuring forms, and linking pages while also allowing me to experiment with creative presentation and design choices.

  1. Improved understanding of HTML5 semantic elements
  2. Experience with form creation and validation basics
  3. Practice in creating readable, navigable web layouts

About the Topic

This project demonstrates the use of semantic HTML and basic web development principles. The pages include structured content with proper headings, paragraphs, lists, and forms, fulfilling the academic objectives of the exercise while also creating a functional and responsive website.

Key Features:

  • Semantic HTML elements for clear structure and accessibility
  • Navigation bar linking all pages for seamless user experience
  • Forms for user interaction, including contact and feedback pages
  • Responsive layout ensuring usability across different screen sizes
